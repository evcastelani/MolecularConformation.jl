64 """
65 IteratorSize(itertype::Type) -> IteratorSize
66
67 Given the type of an iterator, return one of the following values:
68
69 * `SizeUnknown()` if the length (number of elements) cannot be determined in advance.
70 * `HasLength()` if there is a fixed, finite length.
71 * `HasShape{N}()` if there is a known length plus a notion of multidimensional shape (as for an array).
72 In this case `N` should give the number of dimensions, and the [`axes`](@ref) function is valid
73 for the iterator.
74 * `IsInfinite()` if the iterator yields values forever.
75
76 The default value (for iterators that do not define this function) is `HasLength()`.
77 This means that most iterators are assumed to implement [`length`](@ref).
78
79 This trait is generally used to select between algorithms that pre-allocate space for their
80 result, and algorithms that resize their result incrementally.
81
82 ```jldoctest
83 julia> Base.IteratorSize(1:5)
84 Base.HasShape{1}()
85
86 julia> Base.IteratorSize((2,3))
87 Base.HasLength()
88 ```
