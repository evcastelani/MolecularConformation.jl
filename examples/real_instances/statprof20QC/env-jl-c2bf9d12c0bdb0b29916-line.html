137 by zero or more `"var"=>val` arguments `kv`. `withenv` is generally used via the
138 `withenv(kv...) do ... end` syntax. A value of `nothing` can be used to temporarily unset an
139 environment variable (if it is set). When `withenv` returns, the original environment has
140 been restored.
141 """
142 function withenv(f::Function, keyvals::Pair{T}...) where T<:AbstractString
143 old = Dict{T,Any}()

149 finally
